An FSE (Full Site Editing) powered theme expressly optimized for Gutenberg. Developed to be a solid foundational base theme to be used as a stand-alone theme or paired with a child theme for specialized site development.

== Description ==

Proto is a cutting edge next-generation block theme. It is for FSE (Full Site Development) and highly optimized with features to improve the workflow of building a modern styled and fast loading site using the Gutenberg editor.
